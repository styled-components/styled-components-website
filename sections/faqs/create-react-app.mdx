## How do I use styled-components with `create-react-app`?

> **Note:** Create React App is no longer actively maintained. Consider using [Next.js](https://nextjs.org/), [Remix](https://remix.run/), or [Vite](https://vitejs.dev/) for new projects.

styled-components works out of the box with Create React App like any other library.

For advanced features like the [babel plugin](/docs/tooling#babel-plugin), you would need to eject or use tools like [CRACO](https://craco.js.org/). However, for new projects, we recommend using a modern framework with native styled-components support.
